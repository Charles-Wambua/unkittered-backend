## 🔐 AUTHENTICATION EXPLAINED

**Every endpoint (except Auth) requires:**
```
Authorization: Bearer <JWT_TOKEN>
```

**Get token from:**
- `POST /v1/auth/login`
- `POST /v1/auth/register`
- `POST /v1/auth/oauth`

**Default token lifetime:** 7 days

## 📱 CODE EXAMPLE (from TESTING_GUIDE.md)

### Python
```python
import requests

BASE_URL = "http://localhost:8080/v1"

# Register
register_response = requests.post(
    f"{BASE_URL}/auth/register",
    json={
        "email": "test@example.com",
        "password": "SecurePassword123",
        "displayName": "Test User"
    }
)
token = register_response.json()["token"]

# Get profiles
headers = {"Authorization": f"Bearer {token}"}
profiles = requests.get(f"{BASE_URL}/discover", headers=headers).json()["profiles"]

# Like a profile
like_response = requests.post(
    f"{BASE_URL}/likes",
    headers=headers,
    json={"profileId": profiles[0]["id"]}
)
print(f"Is match: {like_response.json()['isMatch']}")
```

### cURL
```bash
# Login
TOKEN=$(curl -X POST http://localhost:8080/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "test@example.com", "password": "pass"}' \
  | grep -o '"token":"[^"]*' | cut -d'"' -f4)

# Get profiles
curl -X GET http://localhost:8080/v1/discover \
  -H "Authorization: Bearer $TOKEN"

# Like a profile
curl -X POST http://localhost:8080/v1/likes \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"profileId": "profile-uuid"}'
```
